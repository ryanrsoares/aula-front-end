HMTL básico - códigos HTML

Aprenda os princápios básicos para confeção de um site em HTML para
que você possa fazer o seu primeiro site.

Motivação.

HTML (abreviação de Hypertext Merkup Linguage) é uma linguagem de
marcação utilizada na estruturação de páginas web. Sua sintaxe é bastante
simples e ,assim como a XML é baseada em tags, que representam os
diversos elementos de uma página, como imagens e links.


Ao acessar uma página wbe através de um navegador, ele é capaz de
interpretar o código HTML e renderizá-lo de forma compreensível para o
usuário final, exibindo textos, botões, et, com as configurações definidas por
meio das diversas tags que essa linguagem dispõe.
